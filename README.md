## 🌍 What is Arabizi?

Arabizi (often called *Franco*, *Arabish*, or *Chat Arabic*) is an orthography that maps Arabic sounds to Latin letters and uses numbers to represent Arabic characters that do not exist in the standard English alphabet.

**Example:**
> **Arabizi:** *Mar7aba ya sadi2i*  
> **Arabic:** *مرحبا يا صديقي*  
> **English:** *Hello, my friend*

### Other Common Names
Depending on the region, you might hear it referred to as:
`Franco Arabic` `Franco` `Arabish` `Arabic Chat Alphabet` `Arabic Latin Script` `Chat Arabic` `Darija Arabizi`

## 📱 Why is Arabizi Used?

Arabizi emerged during the late 1990s and early 2000s out of technological necessity and remains popular today.

**Historical Reasons:**
* Early mobile phones and SMS protocols did not support Arabic script.
* Computer keyboards lacked Arabic layouts.
* English character limits on SMS were more generous.

**Modern Usage:**
Today, it is a stylistic choice heavily used by youth and gamers on platforms like:
* WhatsApp, Messenger, and Discord
* Instagram, TikTok, and X (Twitter)
* In-game chats (Valorant, PUBG, League of Legends)

## 🔠 The Arabizi Alphabet

### Number Replacements
Because the Latin alphabet lacks equivalents for certain Arabic sounds, numbers that visually resemble the Arabic letters are used instead.

| Arabizi | Arabic | Letter Name | Example Word | Meaning |
| :---: | :---: | :--- | :--- | :--- |
| **2** | ء / أ / إ / ق | Hamza / Qaaf* | `insha2allah` | God willing |
| **3** | ع | Ain | `3arab` | Arabs |
| **3'** / **gh** | غ | Ghayn | `3'ali` | Expensive / Dear |
| **5** / **kh** | خ | Khaa | `5aled` | Khaled (Name) |
| **6** | ط | Taa | `6aleb` | Student |
| **6'** | ظ | Zaa | `6'alem` | Unjust |
| **7** | ح | Haa | `mar7aba` | Hello |
| **8** | ق | Qaaf* | `8alb` | Heart |
| **9** | ص | Saad | `9a7` | Correct / Right |
| **9'** | ض | Daad | `9'arb` | Hitting / Strike |

*(Note: The usage of 2 vs. 8 for the letter Qaaf (ق) varies heavily by regional dialect).*

### Latin Letter Mapping

| Latin | Arabic | Notes |
| :---: | :---: | :--- |
| **a / aa** | ا / َ | Short 'a' or long 'aa' (Alif) |
| **b / p** | ب | 'p' is used in loanwords |
| **c** | ك / س | Rarely used alone (mostly 'ch') |
| **d** | د | Daal |
| **dh** | ذ | Dhaal |
| **e / ee** | ي | Short 'e' or long 'ee' |
| **f / v** | ف | 'v' is used in loanwords |
| **g** | ج / ق | Egyptian 'Jeem' or Gulf 'Qaaf' |
| **h** | ه | Haa |
| **i** | ي | Short 'i' |
| **j** | ج | Jeem (Levant/Gulf) |
| **k** | ك | Kaaf |
| **l** | ل | Laam |
| **m** | م | Meem |
| **n** | ن | Noon |
| **o / oo / ou** | و | Waw (varies by French/English influence) |
| **q** | ق | Qaaf |
| **r** | ر | Raa |
| **s** | س | Seen |
| **t** | ت | Taa |
| **w** | و | Waw |
| **x** | كس | Rarely used |
| **y** | ي | Yaa |
| **z** | ز | Zay |
| **zh** | ژ | Persian loanwords |

### Combined Letters (Digraphs)

| Arabizi | Arabic | Sound |
| :---: | :---: | :--- |
| **sh** / **ch** | ش | "sh" as in *shoe* (ch common in N. Africa) |
| **kh** | خ | Guttural "k" (similar to German *Bach*) |
| **gh** | غ | French "r" sound |
| **th** | ث | "th" as in *think* |
| **dh** | ذ | "th" as in *this* |

## 🗺️ Regional Variations

Arabizi is not a monolith; it adapts to local dialects and colonial linguistic influences (English vs. French).

### 🇪🇬 Egypt
* **g** = ج (Pronounced as a hard 'g' like *game*)
* **2** = ق (Pronounced as a glottal stop)
* *Example:* `gamal` → جمل (camel)

### 🇱🇧 🇸🇾 🇯🇴 🇵🇸 Levant (Shami)
* **2** = ق (Pronounced as a glottal stop)
* *Example:* `2alb` → قلب (heart)

### 🇸🇦 🇦🇪 🇰🇼 🇶🇦 Gulf (Khaleeji)
* **g** = ق (Often pronounced as a hard 'g')
* *Example:* `galb` → قلب (heart)

### 🇲🇦 🇩🇿 🇹🇳 North Africa (Darija)
Heavily influenced by French orthography.
* **ch** = ش (Instead of 'sh')
* **ou** = و (Instead of 'oo' or 'u')
* **9** = ق (Very common in Moroccan Darija)
* *Example:* `chokran` → شكرا (thank you)

## ✍️ Transliteration Tips

Because Arabizi is informal, **there is no universal spelling standard.** Vowels are highly flexible.

These are all perfectly valid ways to write "How are you?":
* `Keefak`
* `Kifak`
* `Keefek`
* `Kifek`

Meaning always depends on **context, pronunciation, and the speaker's dialect.** When building parsing tools, ensure your regex or mapping dictionaries account for flexible vowel combinations (e/ee/i and o/oo/u/ou).
